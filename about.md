## 关于博客

欢迎来到我的技术博客！这里是一个专注于程序员技术分享的个人空间，主要涵盖前端开发、后端架构、算法实践、DevOps 运维等领域的内容。

### 博客特色

- **代码优先**：作为技术博客，代码展示是核心。我们使用 Rouge 语法高亮，支持行号显示，提供优秀的代码阅读体验。
- **响应式设计**：无论是在桌面端还是移动端，都能获得良好的阅读体验。
- **图片优化**：所有图片都经过懒加载和响应式处理，确保快速加载。
- **深色/浅色模式**：支持自动切换的深色模式，保护你的眼睛。

### 技术栈

- **构建工具**: [Jekyll](https://jekyllrb.com/) - 静态网站生成器
- **托管**: GitHub Pages / 任意静态托管服务
- **样式**: SCSS + CSS 变量
- **字体**: Inter + JetBrains Mono + Noto Sans SC
