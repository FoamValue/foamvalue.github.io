<div id="com-google-gson">
 <h2>com.google.gson<a class="headerlink" href="#com-google-gson" title="com.google.gson"> ¶ </a></h2>
 <div id="com-google-gson-interface">
  <h3>接口摘要<a class="headerlink" href="#com-google-gson-interface" title="接口摘要"> ¶ </a></h3>
  <ul>
   <li><em> ExclusionStrategy </em>用来决定一个字段或顶层类是否应该序列化或反序列化为 JSON 输出/输入的一部分</li>
   <li><em> FieldNamingStrategy </em>一种机制，提供在 GSON 中自定义字段的命名。</li>
   <li><em> InstanceCreator&lt;T&gt; </em>创建一个没有定义无参构造函数的实例</li>
   <li><em> JsonDeserializationContext </em>反序列化的上下文，在调用方法的过程中传递一个自定义解串器。</li>
   <li><em> JsonDeserializer&lt;T&gt; </em>JSON 的一个自定义解串器</li>
   <li><em> JsonSerializationContext </em>序列化的上下文，在调用序列化方法的过程中传递一个自定义序列。</li>
   <li><em> JsonSerializer&lt;T&gt; </em>一个 JSON 自定义序列</li>
   <li><em> TypeAdapterFactory </em>创建一个相关类型组的类型适配器</li>
  </ul>
 </div>
 <div id="com-google-gson-class">
  <h3>类摘要<a class="headerlink" href="#com-google-gson-class" title="类摘要"> ¶ </a></h3>
  <ul>
   <li><em> FieldAttributes </em>一个存储属性字段的数据对象</li>
   <li><em> Gson </em>GSON 使用的主类</li>
   <li><em> GsonBuilder </em>当你需要设置非默认配置项时，可以使用这个去构建一个 GSON 实例。</li>
   <li><em> JsonArray </em>JSON 数组类型</li>
   <li><em> JsonElement </em>JSON 的一个元素</li>
   <li><em> JsonNull </em>JSON null值</li>
   <li><em> JsonObject </em>JSON 的对象类型</li>
   <li><em> JsonParser </em>一个解析器来解析解析树的 JSON 元素</li>
   <li><em> JsonPrimitive </em>JSON 原始值</li>
   <li><em> JsonStreamParser </em>流解析器,它允许多个异步JsonElements从指定的读者阅读。</li>
   <li><em> TypeAdapter&lt;T&gt; </em>将 java 对象转换成 JSON</li>
  </ul>
 </div>
 <div id="com-google-gson-enum">
  <h3>枚举摘要<a class="headerlink" href="#com-google-gson-enum" title="枚举摘要"> ¶ </a></h3>
  <ul>
   <li><em> FieldNamingPolicy </em>一个枚举,定义了一些标准JSON字段名称的命名约定。</li>
   <li><em> LongSerializationPolicy </em>定义预期的格式，当 long 或 Long 类型被序列化时。</li>
   </ul>
  </div>
 <div id="com-google-gson-exception">
  <h3>异常摘要<a class="headerlink" href="#com-google-gson-exception" title="异常摘要"> ¶ </a></h3>
  <ul>
   <li><em> JsonIOException </em>当 GSON 无法读取一个输入流或写入时，抛出异常。</li>
   <li><em> JsonParseException </em>如果有一个严重的问题,发生在解析Json字符串，抛出异常。</li>
   <li><em> JsonSyntaxException </em>Gson试图读(或写)一个畸形的JSON元素，抛出异常。</li>
   </ul>
  </div>
</div>
<div id="com-google-gson-annotations">
 <h2>com.google.gson.annotations<a class="headerlink" href="#com-google-gson-annotations" title="com.google.gson.annotations"> ¶ </a></h2>
 <div id="com-google-gson-annotations-annotation">
  <h3>注解类型摘要<a class="headerlink" href="#com-google-gson-annotations-annotation" title="注解类型摘要"> ¶ </a></h3>
  <ul>
   <li><em> Expose </em>表示该成员的 JSON 序列化或反序列化应该被暴露</li>
   <li><em> SerializedName </em>提供 name 的值作为字段名称序列化到 JSON</li>
   <li><em> Since </em>表示版本号，因为成员或类型一直存在。</li>
   <li><em> Until </em>表示版本号，直到成员或类型应该存在。</li>
  </ul>
 </div>
</div>

 <h2>com.google.gson.stream<a class="headerlink" href="#com-google-gson-stream" title="com.google.gson.stream"> ¶ </a></h2>
 <div id="com-google-gson-stream-class">
